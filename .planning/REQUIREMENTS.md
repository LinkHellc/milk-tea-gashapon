## v1 需求详情

### 用户身份 AUTH

- [ ] **AUTH-01**: 用户可微信授权登录，获取头像和昵称
- [ ] **AUTH-02**: 用户头像和昵称展示在结果页

### 品牌列表 BRAND

- [ ] **BRAND-01**: 首页展示9个奶茶品牌（一点点、喜茶、霸王茶姬、古茗、茶百道、CoCo、KOI、瑞幸、沪上阿姨），每个品牌显示logo和名字，入口为奶茶杯形态扭蛋机
- [ ] **BRAND-02**: 用户可随时切换品牌

**品牌列表（9个）：**
1. 一点点
2. 喜茶
3. 霸王茶姬
4. 古茗
5. 茶百道
6. CoCo都可
7. KOI
8. 瑞幸
9. 沪上阿姨

### 扭蛋机核心 GASH

- [ ] **GASH-01**: 点击"扭一下"按钮，透明球形胶囊弹出动画
- [ ] **GASH-02**: 云端实现随机抽取概率算法（不可前端暴露）
- [ ] **GASH-03**: 结果页展示奶茶名 + 一句话介绍
- [ ] **GASH-04**: 用户可重新抽取（不满意再扭一次）
- [ ] **GASH-05**: 用户可往当前品牌扭蛋机添加自定义奶茶款型

**每品牌默认奶茶数量：8-12款经典奶茶**

### 分享 SHARE

- [ ] **SHARE-01**: 结果页可生成分享图（奶茶名 + 品牌 + 小程序码）
- [ ] **SHARE-02**: 可分享到微信朋友/朋友圈

### 积分与成就 POINTS / ACHIEVE

- [ ] **POINTS-01**: 每扭一次获得积分，积分累积可查看
- [ ] **ACHIEVE-01**: 成就徽章系统（抽到隐藏款、集齐品牌、连续签到等）

### 历史记录 HISTORY

- [ ] **HISTORY-01**: 记录用户抽取历史，可查看历史结果

## 不做（Out of Scope）

| 排除项 | 原因 |
|--------|------|
| 奶茶百科详情 | 工具定位，不需要内容沉淀 |
| 热量糖分展示 | 非核心需求，增加开发成本 |
| 店铺匹配/LBS | 超出工具定位 |
| 用户评价系统 | 工具即用即走，不需要社交 |
