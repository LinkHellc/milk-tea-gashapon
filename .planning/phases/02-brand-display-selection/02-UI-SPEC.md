## Spacing Scale

All values in rpx, multiples of 4 as required.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4rpx | Icon micro gaps |
| sm | 8rpx | Inline element spacing |
| md | 16rpx | Default element spacing, card internal padding |
| lg | 24rpx | Section padding, between cards |
| xl | 32rpx | Layout gaps between major sections |
| 2xl | 48rpx | Major section breaks |
| 3xl | 64rpx | Page-level padding |

**Exceptions:** None. All Phase 2 spacing adheres to this scale.

---

## Typography

All sizes in rpx (WeChat converts to px). WeChat base font size 20rpx.

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 28rpx | 400 (normal) | 1.5 | General text, brand names in cards |
| Label | 24rpx | 400 (normal) | 1.4 | Captions, hints, secondary text |
| Heading | 36rpx | 600 (semibold) | 1.2 | Page titles |
| Display | 48rpx | 600 (semibold) | 1.2 | Hero text, result display |

**Font weights:** Only 2 weights used: 400 (normal) and 600 (semibold). No bold (700) in Phase 2.

## Component Inventory

### 1. Brand Swiper (Index Page)

**Structure:**
- WeChat `swiper` component, `indicator-dots="{{false}}"` (custom dots)
- 3 items per screen via `previous-margin="30rpx" next-margin="30rpx"`
- `autoplay="{{false}}"` (manual scroll only)
- `circular="{{false}}"` (no infinite loop, shows page count clearly)

**Custom Dot Indicators (Claude's Discretion — circle dots):**
- Container: flex, center, 24rpx below swiper
- Inactive dot: 12rpx circle, #cccccc fill, 8rpx gap
- Active dot: 12rpx circle, #ffb7b2 fill (accent color)

**Swiper item layout:**
- Flex row, space-between, align-items center
- 3 brand cards per swiper page

### 2. Brand Card (Milk Tea Cup Shape, D-09)

**Dimensions:** 220rpx width x 320rpx height

**Visual structure (top to bottom):**
```
┌─────────────────┐
│    Cup Lid      │  ← 40rpx height, darker shade (brand color -15% lightness), rounded top corners
├─────────────────┤
│                 │
│   Brand Logo     │  ← 100rpx height, centered, aspectFit
│                 │
│   Brand Name    │  ← 36rpx, semibold, white text shadow for legibility
│                 │
└─────────────────┘
│   Straw Hole    │  ← 20rpx circle, positioned top-right of body, semi-transparent white
```

**CSS details:**
- Container: border-radius 20rpx, overflow hidden, relative positioning
- Cup lid: background linear-gradient darker, border-radius top only
- Cup body: gradient background using brand color (lighter to darker top-to-bottom), flex column center
- Straw hole: absolute positioned, 20rpx diameter, rgba(255,255,255,0.5) background, top-right area

**Shadow (Claude's Discretion):**
- box-shadow: 0 8rpx 24rpx rgba(0,0,0,0.1)

**Interaction:**
- hover-class: opacity 0.9 (subtle press feedback)
- No active scale transform (maintains milk tea cup illusion)

### 3. Brand Dropdown Selector (Gashapon Page, D-11)

**Structure:**
```
┌─────────────────────────────────────────┐
│  [BrandName]                        ▼  │  ← 72rpx height, white background, border-bottom
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│  一点点                                 │  ← Dropdown panel, white bg, shadow
│  喜茶                                    │
│  霸王茶姬                                │
│  ...                                    │
└─────────────────────────────────────────┘
```

**Dropdown trigger:**
- Height: 72rpx
- Background: #ffffff
- Border-bottom: 1rpx solid #f0f0f0
- Brand name: 32rpx, semibold, #333333
- Arrow: 24rpx, #999999, rotated 0deg (collapsed) / 180deg (expanded)

**Dropdown panel (Claude's Discretion — instant show, no animation):**
- Width: 100% of page
- Max-height: 400rpx, overflow-y scroll
- Background: #ffffff
- Box-shadow: 0 8rpx 24rpx rgba(0,0,0,0.1)
- Item height: 88rpx
- Item text: 28rpx, normal weight, #333333
- Item divider: 1rpx solid #f5f5f5
- Selected item: background #fff5f5 (accent background tint), text #ffb7b2

### 4. Primary CTA Button ("扭一下")

**Dimensions:** 400rpx x 96rpx, border-radius 48rpx (pill shape)

**Visual:**
- Background: #ffb7b2 (accent)
- Text: #ffffff, 36rpx, semibold
- Shadow: 0 8rpx 24rpx rgba(255,183,178,0.4)

**States:**
- Default: as above
- Hover/pressed: opacity 0.85, shadow reduced
- Disabled: background #cccccc, shadow none

### 5. Gashapon Machine (CSS Milk Tea Cup)

**Dimensions:** 400rpx width x 500rpx height

**Structure:**
```
┌─────────────────────────────────┐
│         Cup Lid (darker)        │  ← 60rpx height
├─────────────────────────────────┤
│                                 │
│      Large Brand Logo           │  ← 200rpx, centered
│                                 │
│      Brand Name                 │  ← 48rpx, semibold
│                                 │
│      [ 扭一下 ] button          │  ← CTA inside machine
│                                 │
└─────────────────────────────────┘
```

- Similar milk tea cup visual to brand card but larger and more prominent
- Brand color gradient background
- Contains the "扭一下" CTA button
