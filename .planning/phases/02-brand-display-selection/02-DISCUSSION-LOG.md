## 1. Layout Style

| Option | Description | Selected |
|--------|-------------|----------|
| 3×3 Grid | 每行3个品牌，正方形卡片，紧凑有序 | |
| 2-Column Grid | 每行2个，卡片更大，logo和名字更清晰 | |

## 2. Brand Card Visual

### 品牌卡片形态

| Option | Description | Selected |
|--------|-------------|----------|
| 拟物奶茶杯 | 立式奶茶杯造型，圆角方块+杯盖+吸管孔，真实奶茶杯比例 | ✓ |
| 简化图标+名称 | 每个品牌一个醒目的logo（圆形或圆角方形）+品牌名 | |
| 胶囊机图标+名称 | 透明球形扭蛋机的简化图标，下方显示品牌名 | |

**User's choice:** 拟物奶茶杯

### 品牌配色

| Option | Description | Selected |
|--------|-------------|----------|
| 是，每个品牌独立配色 | 强化品牌识别度，每个卡片用品牌的主视觉色 | ✓ |
| 否，统一用一种温馨奶茶色调 | 所有品牌用同一个色系，整体更统一但品牌区分度低 | |

**User's choice:** 是，每个品牌独立配色

## 3. Brand Switching

| Option | Description | Selected |
|--------|-------------|----------|
| 抽取页顶部下拉选择器 | 抽取页顶部有当前品牌名称+下拉箭头，点击展开品牌列表直接切换 | ✓ |
| 左滑返回首页重新选 | 每次想换品牌就左滑回首页，从品牌列表重新选 | |
| 抽取页底部常驻品牌切换栏 | 抽取页底部有一排品牌小图标，点击直接切换 | |

**User's choice:** 抽取页顶部下拉选择器

## 4. Visual Style

| Option | Description | Selected |
|--------|-------------|----------|
| 马卡龙色系+柔和渐变 | 低饱和度马卡龙色（薄荷绿、樱花粉、奶油黄），背景用柔和渐变 | ✓ |
| 高饱和撞色+现代感 | 更鲜艳的颜色搭配，卡片用撞色，强调年轻活力和戏剧感 | |
| 极简白+品牌色点缀 | 大量留白，背景纯白或浅灰，卡片用品牌色作为唯一亮色 | |

**User's choice:** 马卡龙色系+柔和渐变
