  <name>Task 2: Add onShareAppMessage to result.js</name>
  <files>pages/result/result.js</files>
  <action>
Read the existing pages/result/result.js first.

Add the onShareAppMessage method after onShareTap:

```javascript
  onShareTap() {
    // Share button triggers WeChat native share menu
    // The actual share content is configured in onShareAppMessage
    // Just trigger the share menu
    wx.showShareMenu({
      withShareTicket: true
    });
  },

  onShareAppMessage(res) {
    // Custom share content when user shares via WeChat
    const milkTeaName = this.data.milkTea || '精选奶茶';
    const brandName = this.data.brand || '';
    const description = this.data.description || '';
    
    return {
      title: `我抽到了${brandName}的${milkTeaName}！`,
      desc: description,
      path: `/pages/result/result?brandId=${this.data.milkTea ? '' : ''}&milkTea=${encodeURIComponent(milkTeaName)}&brandName=${encodeURIComponent(brandName)}&description=${encodeURIComponent(description)}`,
      // Image URL will be set by Canvas-generated image in 04-02
      // For now, use default mini-program card
      imageUrl: ''
    };
  }
});
```

Note: The closing of Page({...}) needs to be adjusted. The file currently ends with:
```javascript
  onShareTap() {
    // Implemented in Phase 4
    wx.showToast({ title: '分享功能开发中', icon: 'none' });
  }
});
```

Replace the onShareTap and closing with the new version above.
